# EZShop
Live: https://tayzn.github.io/EZShop/

### Local Setup
- Clone project `git clone https://github.com/Tayzn/EZShop`
- Run `npm i` to install dependencies
- Run `npm start` to start the local server. May take a few minutes to load.
